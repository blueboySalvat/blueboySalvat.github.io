# 攻击类型
主动攻击包括对数据流进行篡改或伪造数据流，可分为四类：伪装、重放、消息篡改和拒绝服务。

被动攻击采取的方法是对传输中的信息进行窃听和监测，主要目标是获得传输的信息。
有两种主要的被动攻击方式：信息收集和流量分析。

# 网络安全属性
机密性：也叫做保密性，指信息不被非授权的用户、实体或过程所获取与利用。
完整性：信息在存储或传输时不被非法修改、破坏、伪造，不发生信息包丢失、乱序等。
可用性：信息与信息系统可被授权实体正常访问并按照需求使用，即授权实体当需要时能够存取所需信息。
真实性：信息的可信度，主要是指对信息所有者或发送者的身份进行确认，对信息的来源进行判断，对伪造来源的信息予以鉴别。
不可否认性：也称为不可抵赖性，即所有参与者都不可能否认或抵赖曾经完成的操作和承诺。
