# 不经意的播种
马斯克能重拾自己儿时的航天梦，付老师能让自己高中时期的创业种子落地生根。去种自己的种子很好，但是有一点要注意的是，无论是那个时间还是现在，都会有无用论者会在你的耳边嚷嚷：还不如多背点单词，还不如去做道题。学习没错，但是别学傻了，别让自己变成一根只会给别人填补的木头棍。开自己的视野，拓自己的广度，让自己长成滋养万物的大树。

# 见世界见众生见自己
听到付老师说自己从出差酒店葛优躺，再到每次出差一定出去体验风土人情的经历的时候，我是有共鸣的。一年之前，我还是那种喜欢窝着的人，唯技术论的人。但是现在我更愿意放下手机，更愿意抽空，出去看人，看车流，看沿街了。慢慢的你会发现有太多太多被你忽略的美，太多太多你错过的人。

# 农村娃的窘境
俞敏洪用 10 年走出自卑，而很多人可能这一辈子都走不出这种自卑。我同样也是农村出身，我深深的体会过这种自卑。最早从青春期开始的容貌自卑，到自己进城时的打扮自卑，从开始明白我和城里娃不一样的时候，我就开始自卑了。其实大家可以从我的网名就可以窥见一斑，如今我并没有完完全全的摆脱那种自卑感，它在内心深处的某些地方，但是我有开始真真切切的去救赎自己了。"幸福的人用童年治愈一生，不幸的人用一生治愈童年"

# 投资自己
不曾了解那些企业家他们的生活是怎样的，只知道他们挺有钱的，应该挺能吃喝玩乐，挺快活的吧？后来才慢慢知道，那些人甚至比我们更努力，更愿意为自己投资。如果你花的钱，是在为自己的未来投资，那么相信你自己的选择就好了，值不值他们说的并不算。（当然还是要谨慎消费啊，别被骗喽）。

# 主动性
当付老师提问一个人成功最关键的是什么时，我相信在座的每个人心中都有了自己的答案，但是我数了数，在现场的大概 60，70 号人中，敢去回答的人不过也就 3，4 个。当然这里面也包括我在内，我回答的时候并非是不紧张的，并非是不忐忑的，并非是不脸红的，但是我还是选择了要去答这个问题，是因为我知道，什么是曝光效应，什么是焦点效应。很多时候都是我从书上看到的这些东西在支撑我踏出那一步。

# 问对的人
在考虑自己要不要创业的时候要去问创业者，在考虑要不要花钱报班投资自己的时候要去问那些报过的人。你去问那些不想也不敢创业的人，就好似是在错误的选项里寻找答案，你找到的答案，你得到的回答也必然是错的，是否定的。只要是你认为正确的，是积极向上且不违法道德法律的事情，去做就好了。不用被那些人左右自己。


# 硬实力决定你能敲开什么门。
硬实力能决定你能敲开什么样的门，如果连门都没有，就不要提什么我要想走多远，爬多高了。

规划好你的技能成长路线，学有余力着要多给自己点一些技能点。对于开发来说，无论是 Java 还是 C，要保持好一定的代码输出，不是说完成多行代码就可以变得多么优秀，而是让初学的自己适应这种编程的感觉。在假期中，在适当的娱乐中，也保持好学习和输出的步伐。


# 软实力决定你能走多远的路
你会发现有些人他们的硬实力非常厉害，但是却屡屡被拒。偏才怪才是企业中不受欢迎的存在，硬实力只占很少，剩下的都是你的软实力，是你的职业素养。妄想用一门技术敲开大厂的门几乎是不可能的。多交流，多看书，多看，多问，多听。做到这些，才能不惧所谓到 35 岁危机，做到这些，才能在中年时不畏年轻人在学习能力上对自己对降维打击。

# 有价值的沟通要及时记录
不知道你是否了解过**艾宾浩斯记忆遗忘规律**，它大概说了个这么个事：

| 时间间隔 | 记忆量 |
| ---- | ---- |
| 刚刚记忆完毕 | 100% |
| 20 分钟之后 | 58.2% |
| 1 个小时之后 | 44.2% |
| 8-9 个小时之后 | 35.8% |
| 1 天后 | 33.7% |
| 2 天后 | 27.8% |
| 6 天后 | 25.4% |
| 一个月后 | 21.1% |

付老师说他们要求员工在开晚会的五分钟之内就要做完记录，我当时还不理解为什么才 5 分钟。直到刚刚我再回顾了一下艾宾浩斯记忆遗忘规律，才发现 5 分钟还真挺合适的。如表所见，我们人类的记忆并不必鱼好多少。其实我们更擅长的是创造，趁着你还想着一点，赶紧记下来吧，让我们从繁重的记忆负担重解脱，用简单的关键词记录就好，这会对你事后回忆有很大的帮助。（忽略我潦草的字迹）。
