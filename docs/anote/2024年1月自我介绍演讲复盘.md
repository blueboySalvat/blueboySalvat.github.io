# 最初期望
流畅的去介绍自己，让同学们对我有一个比较全面的，不同于别人的印象。

# 评估结果 ：
亮点：自己补充了一些即兴的内容
不足：不够自然和流畅，太过紧张

# 分析原因 ：
- 演讲时的说话特别不自然，有种憋气吐字的感觉
- 不必要的词太多
    就是、然后、吧、还有、
- 演讲时的吐字不够清晰，语速偏快
- 没有提前排练，过两遍稿子
- 没有提前熟悉演讲位置
- 没有控制好我可以控制好的因素，比如网络，演讲文稿，这没有控制好，进一步导致了我发挥的失控。
- 欠缺类似的锻炼。肢体语言还不太自然，并且在演讲的时候就意识到自己的舌头在打转了。于是我有意的降低了一下语速。效果并不好，紧张迫使我很难控制我的肌肉。
- 我没有提前向前人沟通。
- 在整个演讲的活动中，可控的因素是：网络、演讲稿，网络和演讲稿完全是可控的。
- 我没有做到最好，我没有提前留意没插网线时的网络情况，在演讲的时候同学们的屏幕没有那么流畅，我没有充分准备好演讲文稿，演示文稿时不够优雅。
# 总结规律 ：
继续保持的：
- 好像没有.....
需要改进增加的：
- 锻炼嘴巴
- 提前熟悉场地
- 提前向前人交流注意事项
- 提前准备好自己的材料
