基本信息（我是谁）

基本信息和面试岗位

各位面试官大家好，我是来自曲阜师范大学网络工程专业的王文鹏，文武双全的文，鹏程万里的鹏，我想要应聘的岗位是 Java 开发工程师。

为什么应聘这个岗位（我为什么来）

说明求职动机以及职业稳定性

其实我和 Java 很早的时候就结下了不解之缘，我高中喜欢的一款游戏---我的世界正是用 Java 开发的，我感觉编程就像玩积木一样好玩，能开动脑筋尽情的创造。 于是我在大学就选了计算机专业，除了完成基础的学业任务外，我还自学了一些内容

  • 我系统的自学了 WEB 安全的相关内容，了解了在 WEB 应用开发时可能出现的常见漏洞，这让我了以后要做出更安全产品的想法。
  • 动手自学了 python 的自动化编程内容，帮同学写了一个图书馆抢座小脚本，这个小脚本从最初的模拟点击版本，再到后面的使用 http 请求的版本。还帮另外一个朋友写了一个青年大学习团委后台自动化统计小脚本，该脚本使用模拟点击，css 元素定位的技术，完成了从网站数据到 excel 数据的导出。一键就可以完成之前需要重复 10 分钟的重复操作。这个经历让我懂了如何应用编程知识来提高效率
  • 负责公司服务器环境的部署、和试运行，为项目上线全做环境准备。在这段经历让我学会了如何部署相关服务，让以及团队交流。

补充：大学也是相关专业，并且在大学对相关的编程知识有一定的了解，所以我的兴趣、经历都和贵公司的岗位比较匹配。同时贵公司的 java 开发的前景广阔，我认为我可以为贵公司带来价值增强的同时，实现个人发展。

说明你为什么能够胜任这个岗位，以及你的相关经历/项目

现在为了让自己更专业，我开始专注于 Java 开发，除了掌握 JavaEE、JavaSE、Mysql、HTML/JavaScript 等基础技术外，我还可以熟练的使用 SpringBoot、Mybatis 等主流框架进行 WEB 应用的开发。使用这些技术我独立开发了鹏程书苑图书系统、极客商城。

除了实现功能，我还比较注重代码的可维护性，我使用了 knife4j 生成 API 文档，以及使用 Git 进行版本控制。同时还使用了前后端校验，在一定成都上保证了服务器不被非法破坏。
